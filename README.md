# AztecMatch

AztecMatch is our SDSU student matching app project. This web app bridges the gap in socializing for SDSU students who commute or have a hard time meeting other students. Our goal is to create a safe and fun platform where students can connect and socialize.

Right now the project includes:
- account registration and login
- profile creation and editing
- simple match suggestions based on major and interests
- basic messaging routes in the backend

The project uses Flask for the backend, SQLite for the database, and HTML, CSS, and JavaScript for the frontend.
