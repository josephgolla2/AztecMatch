# AztecMatch
AztecMatch is a SDSU based relationship building web app. This web app bridges the gap in socializing for SDSU students who commute or have a hard time meeting other students. Our goal is to create a safe and fun platform where students can connect and socialize.
This system handles a python backend that handles users accounts, matching logic and messaging between users. The application store users data and suggest matches based on students profile and interests.
